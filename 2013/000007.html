Hi,

unfortunately I never had access to a Mac, but clip is written in plain C++
and with the Qt Libraries. All file operations and such are implemented by
Qt-Functions. Therefore, Clip _should_ compile on a Mac with Qt without any
problems. If you have the Qt SDK installed, you might just need to run
qmake and than make in the source directory. Or additionally install the Qt
Creator, import the project and click the Run button.

Sorry that I don't have any better advice, but without a access to a Mac, I
could just give you theoretical tips.
